# MaabagerTaks
Aplicación diseñada con Laravel y Vue Cli
Para el despliegue de la aplicacion debe hacer lo siguiente despues de descargar el repositorio:

en la carpeta frontend ejecutar npm install para instalar las librerias necesarias.
* npm run serve unaves intaladas las librerias
* crear una base de datos en MYSQL con el nombre : backendprueba
* en la carpeta backend ejecutar composer update
* configuar el archivo .env para acceder a la base de datos
* en la terminar ejecutar : php artisan migrate
* ejecutar en la trminal : php artisan serve para inicializar el servidor en el backend
* en el navegador donde se despliga la vista de la aplicacion de Vue, crear un usuario
* iniciar sesion
